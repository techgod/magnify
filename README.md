## magnify
#### Find lesser known places while travelling to new cities
A travel app prototype created using Flutter

Works on
- iOS
- Android
- Web

### Screens
- Login
- Explore
- City Places List
- Place Details
- User Reviews
- Contribute a new Review
- Profile
- FAQs & Support

### Instructions
- Clone Repo, ```cd``` into directory and type ```flutter run```
